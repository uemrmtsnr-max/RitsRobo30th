# 立命館大学ロボティクス学科 創立30周年記念イベントサイト

このリポジトリは、イベント向けの静的サイト一式です。

## できること

- トップページ
- 参加登録ページ
- 寄付案内ページ
- 管理画面
- 参加登録データの Supabase 保存
- 管理者だけが閲覧できる登録一覧

## 構成

- `index.html` トップページ
- `register.html` 参加登録ページ
- `donate.html` 寄付案内ページ
- `admin.html` 管理画面
- `assets/` 共通 CSS と JavaScript
- `supabase/schema.sql` データベース定義

## 公開までの手順

### 1. Supabase を作成

1. Supabase で新規プロジェクトを作成します。
2. SQL エディタで `supabase/schema.sql` を実行します。
3. `assets/config.js` の以下を自分の値に置き換えます。
   - `supabaseUrl`
   - `supabaseAnonKey`
   - `adminEmails`
4. Supabase Auth で管理者アカウントを作成します。

### 2. ホスティングに載せる

このサイトは静的ファイルなので、無料の静的ホスティングにそのまま置けます。

候補:

- GitHub Pages
- Cloudflare Pages
- Netlify

### 3. 管理画面の注意

- `admin.html` は Supabase Auth でログインしたうえで、`adminEmails` に含まれるアドレスだけが閲覧できます。
- 登録データは一般公開しません。
- さらに厳密にするなら、Supabase の Auth 設定で許可するリダイレクト URL を公開先ドメインに合わせて設定してください。

## 寄付ページ

寄付の誘導先は立命館大学の公式寄付ページです。

- `https://www.ritsumei.ac.jp/giving/125th/`

必要に応じて、大学側の案内に合わせてリンク先を変更してください。

## 仮文言について

このサイトは仮内容で入れています。
以下の内容は後で差し替えできます。

- 開催日時
- 会場
- プログラム
- 登壇者
- 参加条件
- 問い合わせ先

## 補足

この構成は「無料の静的ホスティング + 無料枠の Supabase」を前提にしています。
もし希望があれば、次の段階で以下も整えられます。

- 申し込み完了メール
- CSV 自動送信
- 参加者管理の検索強化
- 公式デザインに寄せた細かな文言調整
